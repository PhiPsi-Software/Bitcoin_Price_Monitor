The program was written in Python 2.7.

The following Python librarys needs to be installed:

pip install beautifulsoup
pip install colorama
pip install scipy
pip install requests
pip install helper
pip install simplejson
pip install pystray
pip install numpy
pip install matplotlib

Then download and install wxpython (wxPython3.0-win32-py27) from https://www.wxpython.org/download.php.
Do not install wxpython by pip install wxpython!
